Formatação de Strings e Números

Strings

Se tivermos var s = 'JavaScript', podemos formatar essa string com:

  • s.length, para retornar quantos caracteres a string da variável tem;
  • s.toUpperCase(), para que o texto da string passe todo para maiúsculo;
  • s.toLowerCase(), para que o texto da string passe todo para minúsculo.

Números

Se tivermos var num = 1532.398, podemos formatar a quantidade de casas decimais que são exibidas no número e também é possível substituir o ponto por vírgula.

  • Para mudar a quantidade de casas, utiliza-se o comando num.toFixed(2);
  • Já para mudar de ponto para vírgula, é utilizado o comando replace('.', ',') junto com num.tofixed, sendo que ele ficaria num.toFixed(2).replace('.', ','). É importante observar que, neste caso, o dado tipo número passará a ser uma string e que, caso o comando replace() seja utilizado sozinho, ele não irá funcionar a não ser que o a variável seja convertida para uma string.
  • Também é possível fazer uma formatação automática para moeda com o comando
    num.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'})
