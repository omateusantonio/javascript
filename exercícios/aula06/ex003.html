Tipos de dados

Ao usar a concatenação (ou seja, o sinal de mais), o código não sabe diferenciar por si só se a variável é uma string ou um número.

Para que a diferenciação ocorra, é necessário determinar usando o comando Number ou então parseInt (para números inteiros) ou parseFloat (para todos os números reais). É opcional utilizar Number(n) antes dos comandos parse, sendo que ele ficaria Number.parseInt.

Para fazer a conversão de número para string é possível usar o comando String(n) ou então a maneira invertida n.toString()
