Selecionar elementos do DOM

Usando o DOM, é possível selecionar elementos do código por:

  • Marca - getElementsByTagName();
  • ID - getElementById();
  • Nome - getElementsByName();
  • Classe - getElementsByClassName();
  • Seletor (técnica moderna recomendada para CSS) - querySelector() ou querySelectorAll();

Ao usar seletores que não são únicos (todos os acima exceto Id), é NECESSÁRIO selecionar qual das tags se deseja modificar. Para isso, é necessário abrir colchetes para determinar qual das tags será selecionada.
Como por exemplo: getElementsByTagName('p')[2] selecionará a terceira tag <p> da página (a contagem inicia a partir do zero).
